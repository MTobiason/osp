The Oligomer Sequence Package (OSP)

********
Programs
********

Programs in this repository include: 
	(1) The Device Profiler program for analyzing existing networks (abbreviated DevPro).
	(2) The Sequence Evolver program for generating new networks (abbreviated SeqEvo).

*******************
System Requirements
*******************

These programs require Java 8 or newer to be installed. Java can be downloaded for free (https://www.java.com/en/download/).

******************
Compiled Jar Files
******************

Compiled versions of the programs can be created in the “jar” directory by following the instructions in jar/README.txt.

*************
Documentation
*************

Documentation, such as user manuals and tutorials, can be found in the "doc" directory.

***********
Source Code
***********

The source code of the programs can be found in the “src” directory.

********
Examples
********

Example input files for each program can be found in the “examples” directory.
